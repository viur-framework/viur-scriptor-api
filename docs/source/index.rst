Welcome to scriptor's documentation!
====================================

The Scriptor is a client-side scripting environment specifically designed for the Viur framework. With this powerful
component, you can quickly and effectively create scripts to enhance the functionality of your Viur-based applications.

Scriptor provides tools that enable you to develop and execute client-side scripts seamlessly within the Viur framework.
Whether you need to implement complex data manipulation, interactive features, or dynamic user interaction, the
Scriptor empowers you to achieve your goals efficiently. It can also be used to implement backend functionality, which
is especially useful for data validation and manipulation.

Integrating the Scriptor into your Viur-based projects is straightforward. Its intuitive syntax and comprehensive
documentation make it accessible to both seasoned developers and those new to client-side scripting. Unlock the full
potential of the Viur framework by utilizing the Scriptor for your scripting needs.
